# Landing image slots

These are the prepared visual concepts for the reportaAI landing page. Current generated PNG assets are already used by `src/pages/LandingPage.tsx`; when final brand-approved images are available, replace them with optimized `.webp` versions using the same concepts.

- `hero-citizen-reporting.png` - hero image with a citizen using a phone to report an urban issue.
- `citizen-reporting.png` - citizen using a phone to report an urban issue.
- `field-team.png` - field team resolving a municipal service request.
- `city-dashboard.png` - public managers viewing dashboard indicators.
- `city-street.webp` - optional urban street issue identified with photo and location.
- `public-management.webp` - optional public administration team using the platform.
- `smart-city-map.webp` - optional smart city map with geolocated pins.

Use professional, institutional, non-political images only. Do not use city crests, mayoral imagery, campaign material, party colors, or unofficial low-quality photos.
